# vart

Varuna Runtime library.

## Installing

This repository is meant to be used as a `git submodule` for the Varuna compiler,
in the directory src/vart.

```sh
# In compiler root
$ git submodule add https://github.com/varuna-lang/vart.git src/vart
```
